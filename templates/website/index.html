Steps to Create a Django Project

  1. Create a virtual environment using uv venv.
  2. Activate the virtual environment. .venv\Scripts\activate
  3. Install Django using uv pip install django.
  4. Create a Django project using django-admin startproject [folder/project name].
  5. Navigate into the project directory.cd [folder/project name]
  6. Run the development server to verify installation. python manage.py runserver
  7. Create a Django app using python manage.py startapp [app_name].
  8. Register the app in INSTALLED_APPS.
  9. Start building views, templates, and URLs.
